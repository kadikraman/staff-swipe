## Staff swipe
Nifty little UI snippet to show how catering companies could choose employees via a Tinder-like swipecard UI.

Run the app:
```
yarn start
```

Run the tests:
```
yarn test
```

Run the linter:
```
yarn lint
```

Run prettier:
```
yarn prettier
```
